# Projukti Builders Website | প্রযুক্তি বিল্ডার্স

A professional bilingual (Bengali/English) real estate website for Projukti Builders Ltd., Dhaka, Bangladesh.

## Features
- 🌐 Bilingual support (Bengali & English) with language switcher
- 🏠 Property listings with filtering (flats, land shares, commercial)
- 💎 Investment packages showcase (Golden & Diamond packages)
- 📋 Lead capture contact form with inquiry management
- 🗺️ Google Maps integration for project locations
- ⭐ Testimonials & reviews section
- 📱 Mobile-responsive design
- 💬 WhatsApp integration button
- 🔐 Admin panel for inquiry management

## Tech Stack
- React 19 + TypeScript + Tailwind CSS 4
- tRPC 11 + Express 4
- Drizzle ORM + MySQL
- Vite 7
